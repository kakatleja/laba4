Задание 1.

Алгоритм: Переносим первый элемент списка в конец списка, удаляя его с помощью RemoveAt(0) и добавляя в конец через Add, тем самым сохраняя порядок остальных элементов
и корректно перемещая первый элемент в конец.

Тест:

Ввод: 1 2 3 4 → Вывод: 2 3 4 1

Ввод: яблоко груша банан → Вывод: груша банан яблоко

Задание 2.

Алгоритм: Для каждого элемента списка проверяем значения его соседей, считая первый и последний элементы соседями друг для друга, если значения соседей совпадают,
помечаем этот элемент на удаление, после прохода по всему списку удаляем все помеченные элементы, чтобы корректно удалить все элементы, удовлетворяющие условию.

Тест:

Ввод: 1 2 1 2 4 5 → Вывод: 1 2 4 5

Ввод: a b a b a → Вывод: a a

Задание 3.

Алгоритм: Сначала вводим все фабрики магазина, затем для каждого покупателя вводим фабрики, которые он купил, после этого для каждой фабрики проверяем,
куплена ли она всеми покупателями, куплена ли некоторыми, куплена ли никем, и формируем три списка соответственно, выводим их на экран.

Тест:

Фабрики: A B C, Покупатели: 1: A B, 2: B C → Вывод: Все: B, Некоторые: A C, Никем: -

Фабрики: X Y Z, Покупатели: 1:X, 2:Y → Вывод: Все: -, Некоторые: X Y, Никем: Z

Задание 4.

Алгоритм: Читаем текст из файла, с помощью регулярного выражения "\w+" выделяем все слова, приводим их к нижнему регистру, определяем глухие согласные,
создаём множество всех глухих согласных, пересекаем его со всеми буквами нечётных слов, затем исключаем буквы, которые встречаются хотя бы в одном чётном слове,
после этого сортируем результат в алфавитном порядке и выводим на экран.

Тест:

Файл a.txt: "папа б па" → Вывод: g

Файл b.txt: "шар ум шок" → Вывод: ш

Задание 5.

Алгоритм: Для каждой жирности сметаны (15, 20, 25) собираем список всех цен магазинов, проверяем минимальную цену в этом списке, затем считаем количество магазинов,
где цена равна минимальной, формируем итоговый список с тремя числами и выводим его на экран через пробел.

Тест:

Ввод: Магазин1 Улица1 15 2500, Магазин2 Улица2 15 2500, Магазин3 Улица3 20 3000 → Вывод: 2 1 0

Ввод: A S 15 3000, B T 20 3200, C U 25 3100 → Вывод: 1 1 1
